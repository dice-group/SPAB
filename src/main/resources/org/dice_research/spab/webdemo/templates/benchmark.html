Queries have to be separated by at least 2 empty lines.

The benchmark results have to be provided tab-separated (TSV).
This format is often generated, if you mark several cells in spreadsheet applications and copy them, e.g. using [Ctrl]+[C].
The needed fields/columns are:

  • Query-number (in order of the input box above)
  • Triplestore ID (an identificator for the RDF database)
  • Runtime of query on Triplestore
